Display the loaded scripts and styles that are enqueued for optimization or debugging.

== Description ==
In the admin bar, this plugin, adds a display that shows the registered
scripts and styles. This can be helpful for site optimization or debugging.
The plugin adds the 'Display Loaded Scripts' to the admin bar.
Click the 'Display Loaded Scripts button' to see the loaded scripts that
are currently enqueued. Click 'styles' to see the enqueued styles.

== Installation ==

Install the a-admin-loaded-display.zip file like you would any other plugin.

1. Upload `a-admin-loaded-display.zip` to the `Upload Plugin` in the Plugins panel
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The 'Display Loaded Scripts' button is loaded into the admin bar.
